# OOPS Banner App

## Description
A Java-based application demonstrating core Object-Oriented Programming (OOP) concepts through a banner generation system.

## Features
- Demonstrates OOP principles (Encapsulation, Inheritance, Polymorphism)
- Modular class design
- Simple user interaction

## Tech Stack
- Java

## How to Run
1. Compile the program
2. Run the main class
3. Follow console instructions

## Purpose
Built as a learning project to understand OOP fundamentals.
